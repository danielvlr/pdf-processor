╔═══════════════════════════════════════════════════════════════════════════╗
║                         🛠️ OPÇÕES DE INSTALAÇÃO                           ║
╚═══════════════════════════════════════════════════════════════════════════╝

  OPÇÃO 1: Docker (Recomendado - Mais Fácil)
  ────────────────────────────────────────────
    docker-compose up --build
    → Acesse: http://localhost

  OPÇÃO 2: Desenvolvimento Local
  ────────────────────────────────────────────
    npm run install:all
    cd backend && cp .env.example .env && cd ..
    npm run dev
    → Acesse: http://localhost:3000

  OPÇÃO 3: Scripts Automáticos
  ────────────────────────────────────────────
    # Linux/Mac
    chmod +x setup.sh && ./setup.sh

    # Windows
    setup.bat


╔═══════════════════════════════════════════════════════════════════════════╗
║                           💡 COMO USAR                                    ║
╚═══════════════════════════════════════════════════════════════════════════╝

  1. PREPARE ARQUIVOS
     • Crie um ZIP com os PDFs a processar
     • Prepare uma capa (PDF ou imagem)

  2. ACESSE A APLICAÇÃO
     • http://localhost (Docker)
     • http://localhost:3000 (Local)

  3. FAÇA UPLOAD
     • Selecione o ZIP
     • Selecione a capa
     • Configure altura do rodapé (opcional)

  4. PROCESSAR
     • Clique em "Processar PDFs"
     • Aguarde o processamento

  5. DOWNLOAD
     • Download automático do ZIP processado
     • Veja o relatório na tela
